Pour que la .jar fonctionne, il faut créer une BD nommée "reservationmachine".
Voici les informations que le compte de connexion à MySQL devra contenir :
Nom d'utilisateur : 'root'
Pas de mot de passe
Host : localhost
Port 3306

NB : Pour modifier les paramètres de connexion, il faut se rendre dans la classe
'com.reservationmachines.model.BD'

Puis importer les tables au moyen du fichier CREATE_TABLES.sql

NB: Les requêtes respectent les normes de MySQL, donc il faut utiliser MySQL.
